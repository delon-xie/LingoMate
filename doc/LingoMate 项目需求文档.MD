# LingoMate 项目需求文档 (一体化AI英语私教桌面应用)

## 一、产品定义

### 1.1 产品概述
**LingoMate** 是一款面向教育用户的一体化AI英语私教桌面应用，采用"下载即用"的极简部署方案。通过自然语音对话和启发式教学方法，为用户提供专业、沉浸的英语学习体验。

### 1.2 产品名称
**最终选定：LingoMate**

**选择理由：**
- **易用性优先**：`Lingo` 作为常见英文单词，发音简单、拼写直观，符合"降低部署复杂度"的核心目标
- **情感契合**："语言伙伴"的定位与产品"私人外教"理念高度一致
- **市场传播**：易记、易说、易搜索，便于非技术用户间的口碑传播
- **功能匹配**：贴合"情景对话"、"生活化交流"的产品功能设计

**备选方案：**
- LinguaMate：更学术、更正式，适合强调系统化教学的产品定位
- EasyTalk：价值主张明确，但品牌延展性有限

### 1.3 产品定位
**一句话定位：** "像朋友一样自然交谈，像老师一样专业教学的AI英语伙伴"

**定位三层解读：**
1. **关系层**：不是工具，是"伙伴" - 专属、耐心、随时在线
2. **体验层**：不是练习，是"交谈" - 自然、流畅、无压力
3. **价值层**：不是娱乐，是"教学" - 专业、有效、有收获

### 1.4 核心价值主张
**对用户的价值：**
- ⚡ **零门槛启动**：下载安装即用，无需任何技术配置
- 🎯 **专业教学**：基于启发式教学法的AI对话，真正提升语言能力
- 🛡️ **安全环境**：无评判、无压力、可重复练习的私人空间
- 💰 **极致性价比**：一次购买，终身拥有的"私人外教"

**对市场的价值：**
- 填补"AI对话玩具"与"高价真人外教"之间的市场空白
- 提供可量化效果、深度沉浸的AI教育产品新形态

## 二、目标用户

#### 2.1 核心用户画像
| 用户类型 | 核心特征 | 核心痛点与需求 | 典型使用场景 |
| :--- | :--- | :--- | :--- |
| **中学生群体** | 12-18岁，处于语言学习黄金期，学业压力大，对趣味性敏感。 | 1. **应试压力**：需应对中考、高考英语，特别是头疼的口语和听力。<br>2. **开口恐惧**：在课堂和老师同学面前怕犯错，不敢说。<br>3. **方法枯燥**：对死记硬背单词和语法感到厌倦，学习动力不足。<br>4. **资源有限**：难以获得一对一的外教辅导。 | 课后家庭学习、周末备考、寒暑假集中提升。 |
| **大学生群体** | 18-24岁，备考四六级/雅思/托福，或有留学/求职需求。 | 口语考试、学术交流、简历与面试准备。 | 自习室、图书馆、宿舍。 |

该群体是“数字原住民”，对交互体验要求高。他们需要的不是另一个“学习软件”，而是一个**有趣、有安全感、能即时反馈的“智能学伴”**。LingoMate的“无评判环境”和“游戏化情景”恰好击中其“怕丢脸”和“厌枯燥”两大痛点。

## 三、核心体验流程

### 3.1 用户旅程地图
```
发现获取 → 首次启动 → 核心学习 → 长期使用
   ↓           ↓           ↓           ↓
5分钟安装   3分钟设置   沉浸对话   持续进步
```

**阶段A：发现与获取 (目标：零阻力开始)**
- 官网提供清晰的介绍视频和效果演示
- 主流软件站上架，安装包<200MB
- 一键安装，无捆绑，无复杂权限要求

**阶段B：首次启动 (目标：3分钟开聊)**
1. **欢迎向导**：设置昵称、英语水平(A1-C2)
2. **智能检测**：自动识别硬件性能，推荐运行模式
3. **静默部署**：后台下载必要AI组件(明确进度提示)
4. **即刻体验**：完成下载后直接进入欢迎对话

**阶段C：核心学习 (目标：沉浸式进步)**
```
         ┌─────────────┐
         │  选择场景   │
         │ (咖啡店/面试)│
         └──────┬──────┘
                ↓
    ┌─────────────────────┐
    │ 按住说话 → 实时转文字 │
    └──────────┬──────────┘
               ↓
    ┌─────────────────────┐
    │ AI思考 → 流式回复   │
    │ 文字显示 + 语音播放 │
    └──────────┬──────────┘
               ↓
    ┌─────────────────────┐
    │ 双击生词 → 深度教学 │
    │ 解释 → 举例 → 提问 │
    └─────────────────────┘
```


需针对中学生强化以下设计：

1.  **情景设计兼顾趣味与考点**：在6个首发情景中，确保包含如“学校生活”、“兴趣俱乐部”等贴近其生活的情景，同时在对话设计中自然融入中考/高考高频词汇和句型。
2.  **“即点即学”对接课内词汇**：生词本功能可支持**主动导入课本词表**，AI在自由对话中有计划地融合这些词汇进行练习，实现课内外联动。
3.  **成就系统贴合学生心理**：设计“连续打卡学霸”、“场景通关之星”等徽章，并提供可分享的学习成果海报（如“本周我用LingoMate练习了X次口语”），满足其社交展示需求。


**阶段D：长期坚持 (目标：形成习惯)**
- 📊 进度可视化：每周学习时长、新词掌握数
- 🏆 成就系统：连续打卡、场景通关徽章
- 🔄 智能复习：AI在对话中自然复用已学生词
- 📦 内容更新：每月新增情景对话包

## 五、教学体系设计

### 5.1 AI教师行为准则 (System Prompt)
```
角色设定：
你是一位来自北美的专业英语教师，热情、耐心、教学方法高超。

核心教学原则：
1. 只用英语交流（除非学生明确要求且已尝试三次）
2. 引导而非灌输，启发而非告知
3. 保持回复简洁（每次不超过3句话）

单词教学流程：
1. 简单解释：用A1/A2级别词汇解释含义
2. 情景举例：构建贴近学生生活的使用场景
3. 引导应用：提出具体问题，要求学生使用新词

对话风格：
- 自然、友好、鼓励性
- 适当使用"Good job!"、"Interesting!"等正面反馈
- 错误纠正：先肯定，后委婉纠正
```

### 5.2 难度分级体系

#### 1.成人分级
| 等级 | 对应水平 | AI用词难度 | 对话速度 | 教学深度 |
|-----|---------|-----------|---------|---------|
| **初级** | A1-A2 | 基础1000词 | 慢速清晰 | 重点词汇教学 |
| **中级** | B1-B2 | 日常3000词 | 自然语速 | 句型+表达 |
| **高级** | C1-C2 | 学术/专业 | 母语语速 | 逻辑+修辞 |

#### 2.学生分级
| 等级 | 对应学段 | 核心对标 | AI互动重点 |
| :--- | :--- | :--- | :--- |
| **初级** | 初中基础 (A1-A2) | 中考常考日常话题、基础词汇句型。 | 准确性、鼓励开口、建立信心。 |
| **中级** | 高中进阶 (B1) | 高考话题、较长篇幅表达、简单论述。 | 流利度、逻辑组织、文化背景。 |
| **高级** | 拔高/留学准备 (B2) | 雅思/托福口语题型，批判性思维。 | 观点深度、修辞多样性、学术讨论。 |

### 5.3 学习效果保障机制
**即时反馈系统：**
- 语音识别准确率实时显示
- 语法错误高亮提示（可配置开关）
- 发音评分（进阶功能）

**进度追踪：**
- 每周学习报告邮件
- 能力成长曲线图
- 薄弱环节识别与推荐

#### 5.4 与校内学习协同
- **课本同步**：提供按主流教材（如人教版、外研版）单元划分的**补充情景包**，供学生选择。
- **考点强化**：AI可在对话中模拟“英语口试”常见题型（如看图说话、简短问答），并提供即时反馈。
- **学习报告**：生成的学习报告可突出“本周运用的课内词汇数”、“模拟口试流畅度趋势”等，方便学生自我评估和教师（或家长）了解情况。

## 文档总结

LingoMate 的核心创新在于将**专业的启发式教学法**与**极致简单的一体化体验**完美结合。我们不是做一个"更好的AI聊天机器人"，而是创造一个"真正能教会用户英语的数字伙伴"。

项目的成功不取决于技术的先进性，而取决于能否让一个普通的英语学习者，在双击图标后的3分钟内，开始一次有价值的、有收获的英语对话，并愿意明天继续回来。

这既是一个技术产品，更是一个教育产品。所有的技术选择、功能设计、交互细节，都必须服务于"有效教学"和"极致易用"这两个不可妥协的目标。

**最终检验标准：** 一个完全不懂技术的妈妈，能否在儿子的指导下，10分钟内安装并开始和LingoMate进行有意义的英语对话，并在一个月后感觉自己的英语真的有进步。

如果答案是肯定的，那么LingoMate就成功了。
